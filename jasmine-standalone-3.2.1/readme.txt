#Jasmine Standalone Runner Setup

1. Download jasmine standalone zip file  |   https://github.com/jasmine/jasmine/releases
2. Add some spec file inside spec folder |   D:\XXX\XXX\jasmine-standalone-3.2.1\spec
3. Add spec path in SpecRunner.html e.g.
	<script src="spec/spec_1.js"></script>
4. Double click SpecRunner.html
